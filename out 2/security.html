Security & Access Control

Security is a core design principle of uLiquid Market Maker. The system is built to minimize risk while maintaining operational flexibility.

Encrypted Exchange Keys

API keys are encrypted at rest and can only be viewed or modified after re-authentication.

Role-Based Access Control

Granular permissions per workspace for Admins, Operators, and Viewers.

Re-Authentication (OTP)

Sensitive actions require a temporary verification code.

Optional Email 2FA

Additional account protection configurable per user.

CSRF & Origin Protections

Cookie-based authentication hardened with CSRF and origin checks.

Managed VPS Architecture

Dedicated infrastructure provisioned and maintained by uLiquid.

Security Contact

Report vulnerabilities or security concerns to security@uliquid.vip. We respond quickly and coordinate remediation.
